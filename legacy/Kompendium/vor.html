Vorladen id, iannum


Lädt eine Personenanimation im Voraus. So können die kurzen Ladeunterbrechungen während des Spiels vermindert werden, wenn ein flüssiger Ablauf nötig ist. Es werden alle vorhandenen Richtungen der Animationsnummer geladen.


id · ID-Nummer der Person

iannum · Animationsnummer
