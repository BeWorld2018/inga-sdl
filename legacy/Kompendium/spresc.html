SpringeEsc marke


Die Sprungmarke wird sich gemerkt, aber der Sprung selbst noch nicht ausgeführt. Nur wenn der Spieler die ESC-Taste drückt, wird zur angegebenen Sprungmarke gesprungen. Dies ist so lange gültig, bis der Befehl "EscEnde" ausgeführt wird oder ein anderer "SpringeEsc"-Befehl kommt.

Dies kann dafür benutzt werden, um z.B. lange Gespräche abbrechen zu können.


marke · Bezeichnung der Zielsprungmarke (ohne Doppelpunkt am Anfang!).


Beispiel:


SpringeEsc WEITER

Rede 0, 3, "blabla", ""

Rede 0, 3, "blabla", ""

Rede 0, 3, "blabla", ""

Rede 0, 3, "blabla", ""

:WEITER

EscEnde Nach der Sprungmarke muss die ESC-Funktion unbedingt deaktiviert werden!

Rede 0, 3, "Alles klar??", ""
