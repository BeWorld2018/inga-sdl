SpieleModule datei, num, neustart


Lädt und spielt das angegebene Lied (4-Kanal MMD-Format von OctaMED).

ACHTUNG: Hierfür wird "medplayer.library" benötigt! Für kommerzielle und Shareware-Spiele muss man glaube ich ein Lizenzgeld zahlen, wenn man die Library benutzt. Nähere Informationen enthalten die dazugehörigen Dokumentationen.


datei · Name der Datei im Verzeichnis "MED/"

num · Nummer des Liedes, falls es sich um ein Multi-Module handelt, sonst 0

neustart · 0=Falls das Lied schon läuft, so wird es nicht neugestartet.

1=Das Lied wird auf jeden Fall neugestartet.
