LadeSound sid, datei


Lädt eine Sounddatei auf eine bestimmte ID-Nummer. Wenn die ID schon vergeben ist, so wird der alte Sound entfernt, und der neue geladen. Wenn die selbe Datei schon auf der ID ist, so wird sie nicht noch einmal geladen.

Der Sound kann mit dem Befehl "SpieleSound" abgespielt werden.

Achtung! Für die Soundausgabe wird "IngaSound" benötigt, welches auf AHI basiert. Beachte bitte die Lizenzbestimmungen von AHI!


sid · ID-Nummer für den Sound (von 0 bis 7)

datei · Dateiname des Sound im Verzeichnis "Sounds/"
