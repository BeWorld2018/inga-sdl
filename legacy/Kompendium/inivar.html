InitVariable vid, wert


Die gewünschte Variable erhält nur einen neuen Wert, wenn sie noch nicht existiert (ihr also noch nie ein Wert zugewiesen wurde). Falls die Variable schon einen Wert hat, so wird dieser Befehl ignoriert.


vid · Nummer der gewünschten Variable (ohne Doppelkreuz (#) angeben).

wert · Wert, mit dem die Variable ggf. initialisiert wird.
