ListeAbbruch


Nachdem der Spieler ein Objekt oder eine Antwort angeklickt hat, wird das Skript bekanntermaßen nach einem "Liste"- bzw. "Dialogliste"-Befehl fortgeführt. Dabei befindet sich das Programm in einem speziellen Modus, der keine Personenaktionen zulässt. Dies ändert sich erst nach einer bestandenen "Wenn"-Abfrage, oder beim "Stopp"-Befehl.

Mit dem "ListeAbbruch"-Befehl wird ebenfalls der spezielle Modus verlassen, ohne aber die Skriptausführung zu beenden.


Beispiel:

Antwort 1, "eins", 0, 0

Antwort 2, "zwei", 0, 0

Antwort 3, "drei", 0, 0

Dialogliste

WennGesagt 2

Anim 0, 4, 1

Rede 0, 3, "Zwei war richtig!", ""

Stopp

Abschluss

ListeAbbruch

Anim 0, 5, 1

Rede 0, 3, "Das war falsch.", "" *

Stopp

* Beim Warten auf die Vollendung von "Anim" würde ohne "ListeAbbruch" eine Endlosschleife entstehen, da "Anim" im speziellen Modus nicht ausgeführt werden kann.
