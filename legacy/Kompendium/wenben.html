Die Skriptzeilen zwischen dem "WennBenutzt" und dem dazugehörigen "Abschluss" werden nur ausgeführt, wenn ein bestimmtes Element im Spiel benutzt worden ist (linker Mausklick). Diese Abfrage wird in der Aktionsliste (nach dem Befehl "Liste") genutzt.

Gibt man als ID eine Null an, so gilt die Abfrage immer als wahr, falls irgendetwas benutzt wurde.
