Rede id, iannum, text, snd

Abkürzung: R


Gibt einer Person den Befehl zu reden (bzw. den Text einzublenden). Falls schon eine (oder die selbe) Person redet, wird so lange gewartet, bis sie fertig ist.

Achtung! Für die optionale Sprachausgabe wird "IngaSound" benötigt, welches auf AHI basiert. Beachte bitte die Lizenzbestimmungen von AHI!


id · ID-Nummer einer Person.

iannum · Nummer der Personenanimation, die beim Reden gezeigt wird.

text · Eine Textzeile, die beim Reden angezeigt wird.

snd · Dateiname eines Sounds im Verzeichnis "Sounds/", der beim Reden abgespielt wird, oder eine leere Zeichenkette (""), wenn kein Sound existiert.
