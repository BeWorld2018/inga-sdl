Abkürzung: WA ... Ab


Die Skriptzeilen zwischen dem "WennAngesehen" und dem dazugehörigen "Abschluss" werden nur ausgeführt, wenn ein bestimmtes Element im Spiel angesehen worden ist (rechter Mausklick). Diese Abfrage wird in der Aktionsliste (nach dem Befehl "Liste") genutzt.

Gibt man als ID eine Null an, so gilt die Abfrage immer als wahr, falls irgendetwas angesehen wurde.


id · ID-Nummer eines Elements (oder Inventar).
