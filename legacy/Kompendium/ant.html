Antwort id, text, wert, wert


Dieser Befehl schaltet das Spiel in den Multiple-Choice-Modus und fügt eine Auswahlmöglichkeit (Antwort) hinzu, allerdings nur, wenn die beiden Werte übereinstimmen. Soll es die Antwort immer geben, so kann man beide Werte z.B. auf Null setzen. Wenn die Antwort aber nur unter einer bestimmten Voraussetzung möglich sein soll, kann man einen Wert durch eine Variable ersetzen. Die Antwort wird dann nur gezeigt, wenn der Variablenwert mit dem anderen Wert übereinstimmt. Um mehrere Antwortmöglichkeiten zu geben, muss man einfach mehrere "Antwort"-Befehle untereinander schreiben. Beendet wird so eine Liste mit dem Befehl "Dialogliste".

Dieses Prinzip wird hauptsächlich für adventuretypische Dialoge verwendet. Eine Antwort wird aber nicht automatisch von einer Person gesagt, wenn man sie auswählt! Dies muss zusätzlich in den Abfragen programmiert werden.


id · ID-Nummer der Antwort.

text · Die Antwort an sich als Text.

2 x wert · Werte müssen übereinstimmen, damit die Antwort gezeigt wird.
